3.3 Garbage

broncos_gordon_url <- "https://www.pro-football-reference.com/players/G/GordMe00.htm"

broncos_gordon_url %>%
  # read the HTML
  read_html(.) %>%
  # isolate the node containing the HTML table
  html_node(., css = '#stats') %>%
  # parse the html table
  html_table(., fill = TRUE) %>%
  # make the first row of the table column headers and clean up column names
  row_to_names(., row_number = 1) %>%
  clean_names()

Again, these data need some work before they are useful, but we will defer to the next chapter. Right now, let’s focus on taking this methodology and applying it for every player on the 2020 Denver Broncos team who recorded statistics in rushing or receiving categories. We’ll need the web address for each player in the Rushing and Receiving table. Thankfully, this information must exist within the node which contains the table. Otherwise, the hyperlinks would not exist.

We’ll start by isolating the node containing the Rushing and Receiving statistics before scraping the web addresses.

df <- broncos_url %>%
  read_html() %>%
  html_node(., css = '#all_rushing_and_receiving') %>%
  html_nodes(., xpath = 'comment()') %>%
  html_text() %>%
  read_html() %>%
  html_table()
