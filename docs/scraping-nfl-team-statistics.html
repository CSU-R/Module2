3.1 Scraping NFL Team Statistics

The first consideration when scraping data is from where to scrape. Finding the right source often starts with a Google search. One common, reliable archive for football statistics is Pro Football Reference. We will utilize this website for scraping NFL statistics. Ultimately, we hope to supplement the betting information obtained from Bovada’s API with infromation obtained from Pro Football Reference.

The second consideration when scraping data is scaling down the problem. While we may wish to scrape team statistics for all NFL teams across multiple years, it is beneficial to start by developing code to scrape information for one team in a given year. Once you have solved this problem, you can generalize your code to account for various teams and years.
